# Sistema de Login

Sistema de autenticação simples desenvolvido em C.

## Funcionalidades

- Entrada de usuário e senha via terminal
- Verificação usando `strcmp()`
- Mensagem de acesso liberado ou negado

## Como executar

```bash
gcc -o login "SISTEMA DE LOGIN.c"
./login
```

## Exemplo

```
==== SISTEMA DE LOGIN ====

Digite seu usuario: admin
Digite sua senha: ********

Acesso liberado!
```
