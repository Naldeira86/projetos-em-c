## 🚀 Como Compilar

Cada projeto pode ser compilado individualmente com GCC:

```bash
# Calculadora IMC
gcc -o imc "Calculadora IMC/CALCULADORA IMC.c"
./imc

# Sistema de Login
gcc -o login "Sistema de login/SISTEMA DE LOGIN.c"
./login
```
